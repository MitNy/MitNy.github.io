[PCTF2019] docker

지난주에 참여했던 PliadCTF 2019의 docker 문제이다. Misc 10pts짜리 문제.

도커 설치 후 docker pull whowouldeverguessthis/public를 입력하면 문제 이미지를 받을 수 있다.

컨테이너 생성 후 접속해보니 flag 파일이 있어서 flag 파일을 읽으면 되는줄 알았다.

그런데 역시나 쉽게 플래그를 주지 않았다 ㅠ0ㅠ 대신 docker에는 이미지의 히스토리를 볼 수 있는 docker image history 명령어가 있다. 이 명령어를 통해 플래그를 얻을 수 있었다!!

--no-trunc 옵션은 출력이 길어서 잘린 부분을 모두 출력해주는 것이다.

history 명령어는 평소에 자주 쓰지 않아서 삽질을 했는데 이번 기회에 알게 되었으니 앞으로 많이 사용할 것 같다 :)
