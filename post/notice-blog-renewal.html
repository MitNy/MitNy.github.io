블로그 리뉴얼

이번 블로그까지 포함하면 총 3번의 블로그 변경이 있었다.
첫번째는 Tistory,
두번째는 현재와 같은 Github pages였지만 Jekyll로 만들어진 블로그 중 맘에 드는 테마를 포크해서 써왔다.
그러다보니 커스텀도 어려웠고.. 최근에 다시 블로그 포스팅을 하고싶어져서 공부중인 Next.js로 새로 만들기로 했다.
흰 화면에서부터 하나씩 붙여나가니 훨씬 재미도 있고 내 마음대로 구조를 바꿀 수 있어서 좋다.

예전 블로그 포스팅에 대해서..

블로그 리뉴얼을 하면서 포스팅 성격이 기존과는 조금 달라질 것 같아서 기존 글들은 새 블로그에 남겨두지 않으려고 했었다.
그러다가 Google Search Console에서 예전 블로그 기록들을 보니
몇몇 분들께서 참고용으로 예전 포스팅을 링크 해두신게 있어서 그 분들을 위해서도 그렇고
어떤 경로에서든 예전 포스팅을 찾으시는 분들을 위해서 현재 블로그에서도 접근할 수 있도록 하기로 했다.
업무나 공부에 어려움이 있어 검색해서 찾은 글을 눌렀더니 404 Not Found가 뜨는 경험은 누구에게도 좋지 않을 것이다...
부족한 글들이었지만 찾는 분들이 계심에 감사하다. 조금이라도 도움이 되셨다면 다행이고 새로 출발하는 블로그도 잘 봐주셨으면 좋겠다.
기존에는 보안, 웹 개발 등등 여러 종류의 글을 올렸었는데 새 블로그에서는 프론트엔드 개발 위주로 올려보려고 한다.

  • 신규 포스팅 경로: https://mitny.github.io/post/{title} EX) http://mitny.github.io/post/notice-blog-renewal
  • 예전 포스팅 경로: https://mitny.github.io/articles/{yyyy-mm}/{title} EX) https://mitny.github.io/articles/2016-08/gdb-command

추후에 예전 포스팅들도 메뉴로 빼서 리스팅 할 예정이지만 어떤 글들이 있었는지 궁금하시다면 구글의 site 명령어로도 보실 수 있습니다.
(일부 포스팅은 복구하지 않았습니다.)
